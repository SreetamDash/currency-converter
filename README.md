# 💱 Currency Converter — Day 6 of #10DaysOfWebDev

A currency converter web app that fetches live exchange rates and converts between major world currencies instantly!


## ✨ Features

- Convert between USD, INR, EUR, GBP, JPY
- Live exchange rates using a free API
- Input validation for invalid amounts
- Error handling if API fails
- Clean green themed UI

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

---

## 🔗 API Used

- [Open Exchange Rates API](https://open.er-api.com) — free API for live currency exchange rates

---

## 📁 Project Structure

```
currency-converter/
├── index.html
├── style.css
└── script.js
```

---

## 🚀 How to Run

1. Download or clone the files
2. Open `index.html` in your browser
3. Enter an amount, select currencies and click Convert!

---

## 💡 What I Learned

- Using `async / await` with `fetch()`
- How to read and use API response data
- Working with `<select>` dropdowns in JavaScript
- Handling errors gracefully with `try / catch`
- Formatting numbers with `.toFixed(2)`
